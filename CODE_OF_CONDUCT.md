# Code of Conduct

This is an open-source project. Contributions are welcome.

## Expected Behavior

- Communicate professionally and on topic
- Criticize code, not people
- Accept constructive feedback
- For large changes — open an issue first to discuss

## Unacceptable Behavior

- Insults and personal attacks
- Spam and off-topic
- Publishing others' personal data

## Enforcement

Violations can be reported via Issues or directly to the maintainer. The maintainer reserves the right to close any issue or PR without explanation if this Code of Conduct is violated.
